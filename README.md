# Homelab

My homelab config Nix.

## Tools

- Nix and NixOS for package management and system configuration.
- Deploy-rs for managing nix deployments.
- K3s for running Kubernetes clusters.
- Kubenix for managing Kubernetes clusters.
- Rook-Ceph for storage management.
- Sops-nix for secrets management.
- Prometheus and Grafana for monitoring and visualization.
