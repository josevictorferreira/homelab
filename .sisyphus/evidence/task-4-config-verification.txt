# Task 4: Configure Synapse media_storage_providers for S3/RGW

## Verification Evidence

### 1. media_storage_providers Config (ConfigMap)

Location: `.k8s/apps/matrix.yaml` (ConfigMap `synapse-matrix-synapse`)

```yaml
media_storage_providers:
- config:
    bucket: matrix-synapse-media
    endpoint_url: http://rook-ceph-rgw-ceph-objectstore.rook-ceph.svc.cluster.local
    prefix: synapse/
    region_name: us-east-1
    request_checksum_calculation: when_required
    response_checksum_validation: when_required
  module: s3_storage_provider.S3StorageProviderBackend
  store_local: true
  store_remote: true
  store_synchronous: false
```

Synapse is now configured to:
